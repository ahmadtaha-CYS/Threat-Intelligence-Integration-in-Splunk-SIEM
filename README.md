# Threat Intelligence Integration in Splunk SIEM

University project focused on integrating threat intelligence into Splunk SIEM for detecting malicious activity.

## Project Files
- `SOC-report.pdf` — Full project report
- `SOC-presentation.pdf` — Project presentation slides

## Project Summary
This project integrates threat intelligence indicators into Splunk SIEM to detect malicious IPs, domains, URLs, and file hashes. It includes IOC collection, enrichment, simulated log generation, correlation rules, and alerting.

## Tools Used
- Splunk Enterprise
- AlienVault OTX
- Python
